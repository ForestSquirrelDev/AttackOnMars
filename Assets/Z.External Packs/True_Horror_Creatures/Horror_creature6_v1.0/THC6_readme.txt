NighMare Creature #6 readme.

Controls:

To change the mode press:
1 - still mode (walk movement)
2 - combat mode (run movement)

(most actions available in the combat mode)

moving - directional arrows

mouse 1 - attack1
mouse 2 - attack2
mouse 3 - attack3
space - jump
c - roar (howl)
u - hit1/hit2 randomly
y - power hit
i - death/falling
k - rising
o - death2
